# 掘金阅读模式插件

## 项目简介
这是一个简单的Chrome插件, 安装后, 点击按钮可以在掘金文章页面点击进入阅读模式,去除多余页面结构, 给你更清爽的阅读体验.

## 项目功能
* [阅读模式] 调整页面宽度为 1280px, 隐藏侧边栏, 隐藏广告 Banner, 隐藏相关推荐
* [宽屏模式] 调整内容区域宽度, 不对现有页面结构做改动 [未完成]
* [自定义模式] 可手动输入数值, 控制页面宽度 [未完成]
